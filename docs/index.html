    Welcome to the More-Recipes API documentation! This API powers the More-Recipes web app, all .You can use our API to access More-Recipes API endpoints, which can get information on various recipes and users in our database.

    Making Request

    Upon signing up or logging in with the More-Recipes API an authorization token is generated and it can be used to access other resources from the More-Recipes API.

    The generated token expires within 24 hours after it was generated.

    Errors

    The More-Recipes API uses the following error codes:

    Error Code Meaning
    400 Bad Request -- Your request sucks.
    401 Unauthorized -- Your API key is wrong.
    403 Forbidden -- The recipe requested is hidden for administrators only.
    404 Not Found -- The specified recipe could not be found.
    409 Conflict -- Username or email already taken
    500 Internal Server Error -- We had a problem with our server. Try again later.
    503 Service Unavailable -- We're temporarily offline for maintenance. Please try again later.
